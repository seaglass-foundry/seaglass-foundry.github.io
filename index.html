SwingToPDF

True vector PDF output from Java Swing components

Convert any JComponent tree into genuine vector PDF. Text stays selectable, graphics stay sharp, and file sizes stay small. Built on Apache PDFBox.

Java 17+ · AGPL-3.0 or commercial · v1.1.0

Worldwind Reforged

A modern Java 17+ revival of NASA WorldWind

A 3D globe visualization toolkit for Java — shader-based rendering, GPU-accelerated surface shapes, and full backwards compatibility with the original WorldWind Java API. Open source under our Cannonforge banner.
